## 4. Cron operational health — ✅ ALL RUNS GREEN

10 most recent runs of `jobber-poll.yml`: **10/10 success**, average duration ~60-90s.

### 4.1 Cron actual cadence ≠ scheduled cadence — 🟡 MEDIUM

Configured: `*/5 * * * *` (every 5 min).
Observed cadence (last 24h): runs spaced **1–3 hours apart**, not 5 min.

This is GitHub Actions' [well-documented cron jitter](https://github.blog/changelog/2019-11-08-github-actions-now-supports-scheduled-events/) — they delay scheduled runs under load. Effects:
- Data freshness floor is 1–3 hours, not the 5 min you might expect
- For an ops business that batches dispatch decisions, this is fine
- If tighter SLO needed: move cron to Cloudflare Workers Cron Triggers (free, fires on time)

Now that the repo is public, you could also bump the cron to `*/2 * * * *` — GitHub still won't fire that often, but you get more chances.

## 6. Issues to address (priority order)

### 🟡 MEDIUM 1 — Visit handler bug: NULL visit_date

**File:** `supabase/functions/webhook-jobber/index.ts` (handleVisit)
**Issue:** When Jobber returns a visit without `startAt`, our handler builds `visit_date: v.startAt ? v.startAt.slice(0,10) : null` but `visit_date` is NOT NULL in the schema. Insert/update fails for ~33 visits.

**Fix options:**
- (A) Skip visits with no startAt entirely (`if (!v.startAt) return { entity_id: 0 }`)
- (B) Fall back to `endAt` or `completedAt` for visit_date
- (C) Make `visits.visit_date` nullable

Recommend B with C as fallback — preserves the row.

### 🟡 MEDIUM 2 — webhook_events_log retention

Adding ~2k rows/day. Schedule a daily DELETE of rows older than 30 days. ~5 lines of SQL + a cron entry.

### 🟢 LOW 3 — Cron jitter under-fires

Switch to `*/2 * * * *` since public Actions is free, OR migrate to Cloudflare Workers Cron Triggers if jitter matters.

### 🟢 LOW 4 — Stale-cache retries (largely cleared)

11 dead visits + 1 dead client cleared this audit. A few more will accumulate over time. Either:
- Add a "max retries before quarantine" counter to `raw.jobber_pull_*`
- Add a similar daily cleanup job
